# Promptica — лендинг

Лендинг AI-агентства Promptica. Дизайн в Pencil (`.pen`), код генерируется из дизайна.

## Структура
- `design/` — `.pen`-файлы (источник дизайна, версионируются в git)
- `src/` — сгенерированный код (появится позже)

## Заметки к реализации
- **Hero — вращающийся заголовок (typewriter).** Фиксированная часть «ИИ-агент, который» + сменный сегмент акцентом с мигающей кареткой. В дизайне сегмент статичен; в коде анимируется эффектом «печатает/стирает». Набор фраз:
  1. обрабатывает заявки
  2. ведёт клиента в CRM
  3. разбирает почту
  4. готовит КП
  5. работает по вашим правилам
- **Анимация печатает/стирает:**
  - Печать: символы фразы появляются по одному, ~50–80 мс на символ.
  - Пауза на готовой фразе ~1.5–2 с.
  - Стирание: символы убираются по одному, ~30–40 мс на символ (быстрее печати).
  - Переход к следующей фразе, по кругу.
  - Каретка (`Caret`): мигает ~1 Гц (видна/скрыта по 500 мс); во время печати/стирания может не мигать.
  - Доступность: при `prefers-reduced-motion` анимацию отключить — показывать одну фразу статично.
